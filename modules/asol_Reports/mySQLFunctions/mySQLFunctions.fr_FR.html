Flow Control

Flow Control Operators returns a value depending on the result of a given condition.
OperatorDescriptionExample
IF(Cmp_Expr, TRUE_Val, FALSE_Val)If Cmp_Expr equals TRUE returns TRUE_Val, else returns FALSE_ValIF(DAY(date_entered) > 15, '1st', '2nd')
IFNULL(Cmp_Expr, NULL_Val)If Cmp_Expr equals NULL returns NULL_Val, else returns Cmp_ExprIFNULL(type, 'Empty Value')
NULLIF(Cmp1, Cmp2)If (Cmp1 == Cmp2) returns NULL, else returns Cmp1NULLIF(date_entered, date_modified)
